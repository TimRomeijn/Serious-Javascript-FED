Advanced Frontend Development Concept

Voor mijn eindopdracht wil ik een vergelijkbare webapp als onderstaande hero gallery maken van de game Overwatch. Aangezien er nog geen api is zal ik een eigen api maken waarin ik standaard data per hero zal opslaan. Ik zal het beperken tot 5 hero’s zodat het werk te doen blijft. 

De hoofdmenuview zal een knop hebben die je naar de gallery brengt. Vervolgens zullen er 5 hero’s staan waar je op kan klikken en deze geven per hero ook een aparte view met een overzicht van de data die bij de specifiek hero hoort.

Ik zal gebruik maken van het backbone framework om de JS af te handelen en php/json voor de api.
